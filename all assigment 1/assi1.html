Intoduction to HTML

HTML(Hypertext Markup Language) is used to create document on the World Wide Web. It is simply a collection of certain keywords called "Tags" that are helpful in writting the document to be displayed using the browser on Internet. HTML was developed by Tim Berners-Leein 1992.
